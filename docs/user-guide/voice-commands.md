# Voice Command Reference

VibeMind understands natural language in German and English. Below are the recognized command patterns grouped by space.

## Bubble Management

| Command (DE) | Command (EN) | Action |
|-------------|-------------|--------|
| "Erstelle Bubble [Name]" | "Create bubble [Name]" | Create a new bubble |
| "Geh in [Name]" | "Go into [Name]" | Navigate into bubble |
| "Zurueck" / "Raus" | "Back" / "Exit" | Exit current bubble |
| "Zeig mir meine Bubbles" | "Show my bubbles" | List all bubbles |
| "Loesche Bubble [Name]" | "Delete bubble [Name]" | Delete a bubble |
| "Suche Bubble [Query]" | "Find bubble [Query]" | Search bubbles |

## Tips

- **Natural language works** — You don't need exact phrases. "Mach mal ne Bubble Marketing" works like "Erstelle Bubble Marketing"
- **Context-aware** — "Loesch das" (delete that) resolves from conversation context
- **Bilingual** — Mix German and English freely
